Let vs Var

  • var is Function scope, let is block scoped
  • var allows redeclaration
    var a = 5
    var a = 6 // Allowed
    let a = 5
    let a = 6 // Compilation Error
  • Hoisting occours in var
    console.log(var1); //Returns Undefined
    var var1 = 2;
    console.log(var2); //Compilation Error
    let var2 = 2;
